I tried to find following information in documentation of properties using some regex magic
  - result datatype name for each property whose datatype starts with OM_
  - observed property name for each property whose datatype starts with OM_
  - unit of measurement for each property whose datatype is OM_Measurement

Below is list of properties that misses some information.

Just for info, here is the magic:
- re.compile('^.*result (?:should be |expressed as |is )?(?:of type|an|a) (?P<result_datatype_name>\\w+).*$', re.IGNORECASE | re.DOTALL)
- re.compile('^.*observedproperty *= *(?P<observed_property>\\w+).*$', re.IGNORECASE | re.DOTALL)
- re.compile('^.*UoM(?: in | is |: )(?P<unit_of_measurement>(?:\\w|/)+).*$', re.DOTALL)
